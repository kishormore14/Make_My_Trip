# make_my_trip
## Frontend
- React
- Material UI
- Styled- components
## Backend
- Node Js
- Express
- MongoDB
- Nodemailer

## Run Locally

Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd backend/frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
